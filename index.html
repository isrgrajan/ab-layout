📌 About

AB Layout is a lightweight Microsoft Word Add-in that enables advocates to apply court-specific formatting instantly with one click.

🚀 Features

  • 🏛 One-click court formatting
  • 📄 High Court, District Court & Affidavit layouts
  • 🔁 Undo support
  • ⚙ Preset-based architecture
  • 🌐 Live updates via GitHub
  • 💻 Works on existing documents

📥 Installation

  1. Download the add-in
  2. Open Microsoft Word
  3. Go to Insert → Add-ins → Upload My Add-in
  4. Select manifest.xml
